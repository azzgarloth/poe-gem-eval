# PoE Gem Evaluation

A web app that evaluates Path of Exile gems against the live cost of a 20x Gemcutter's Prism bundle.

## Quick Start

1. Download and install [XAMPP](https://www.apachefriends.org/pl/download.html).
2. Download this project and extract it into your XAMPP installation at: `C:\xampp\htdocs\projects\poe-gemEval`.
3. Open the XAMPP Control Panel and click **Start** next to the **Apache** module.
4. Open your web browser and go to: `http://localhost/projects/poe-gemEval/`
